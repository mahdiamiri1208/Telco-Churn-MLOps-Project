# screenshots/

این پوشه برای اسکرین‌شات‌های واقعی خودتان از اجرای پروژه است. طبق انتظارات پروژه،
پیشنهاد می‌شود حداقل این موارد را اینجا قرار دهید (نام‌گذاری پیشنهادی):

- `01_pipeline_run.png` — خروجی ترمینال از اجرای `python run_pipeline.py`
- `02_mlflow_ui.png` — صفحه اصلی MLflow با لیست run ها
- `03_mlflow_compare.png` — مقایسه چند run/مدل در MLflow
- `04_models_folder.png` — محتوای پوشه `models/` (best_model.joblib, confusion matrices, ...)
- `05_git_log.png` — خروجی `git log --oneline --graph`
- `06_github_repo.png` — صفحه Repository در گیت‌هاب با تاریخچه Commit
- `07_docker_build.png` — خروجی `docker build`
- `08_docker_run.png` — خروجی `docker run` (کانتینر در حال اجرا)
- `09_api_health.png` — پاسخ `GET /health`
- `10_api_predict.png` — پاسخ `POST /predict`

این اسکرین‌شات‌ها را خودتان از اجرای واقعی پروژه روی سیستم خودتان تهیه کنید تا
مستندی معتبر و متعلق به کار خودتان باشد.
